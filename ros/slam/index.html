slam

Core SLAM node that subscribes sensor data, runs the SLAM pipeline, and publishes SLAM results as ROS messages.

Subscribed topics

Section titled “Subscribed topics”
ParameterMessage typeDefaultDescription
sub.cloud_topicsensor_msgs/PointCloud2/cloudPoint cloud topic (required)
sub.imu_topicsensor_msgs/Imu""IMU measurements (optional)
sub.absolute_position_topicgeometry_msgs/PoseWithCovarianceStamped""Absolute position measurements, e.g. from gnss_to_utm_pose (optional)
sub.visualization_image_topicsensor_msgs/Image""Image for visualization (optional)

Set a topic to "" to disable the subscription.

Published topics

Section titled “Published topics”
ParameterMessage typeDefaultDescription
pub.processed_cloud_topicsensor_msgs/PointCloud2/slamlab/processed_cloudCurrent filtered and deskewed point cloud in sensor frame
pub.deskewed_cloud_topicsensor_msgs/PointCloud2""Current original (i.e. non-filtered) and deskewed point cloud in map frame
pub.local_map_cloud_topicsensor_msgs/PointCloud2/slamlab/local_mapVoxel-plane centroids of local map
pub.global_map_cloud_topicsensor_msgs/PointCloud2/slamlab/global_mapAccumulated global map
pub.correspondences_cloud_topicsensor_msgs/PointCloud2/slamlab/correspondencesICP correspondences between processed point cloud and local map
pub.path_topicnav_msgs/Path/slamlab/pathEstimated trajectory
pub.cloud_queue_size_topicstd_msgs/UInt32""Current point cloud queue size

Set a topic to "" to disable the publication.

TF parameters

Section titled “TF parameters”
ParameterTypeDefaultDescription
pub.base_frame_tfbooltruePublish mapTbase transform
map_frame_idstringslamlab_mapID of map frame
base_frame_idstringslamlab_baseID of base frame
get_initial_base_pose_from_tfboolfalseGet initial mapTbase transform from TF tree
initial_base_frame_idstringslamlab_base_initialID of base frame (only used if get_initial_base_pose_from_tf is true)
get_static_transforms_from_tfboolfalseOverride static transforms between frames (e.g. between IMU and base frame) from TF tree
